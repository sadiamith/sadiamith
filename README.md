### Hi, I'm Sadi.

I build tools between messy real-world data and the humans who act on it. Based in Calgary, Alberta.

At [Crbon Labs](https://crbonlabs.com) I'm building the data platform behind Canada's first orphan-well plugging carbon credit issuance — event-driven pipelines on AWS, real-time regulator dashboards, and **CarbonFree AI**, a domain-specific 26B-parameter LLM fine-tuned with QLoRA and served through vLLM.

#### Building in the open

- [`@crbonfree/sdk`](https://www.npmjs.com/package/@crbonfree/sdk) — TypeScript SDK for the CrbonFree carbon-offset API
- [`crbonfree`](https://pypi.org/project/crbonfree/) — Python sibling of the same SDK
- [`@crbonfree/mcp`](https://www.npmjs.com/package/@crbonfree/mcp) — MCP server so AI agents can query and offset the carbon cost of their inference in natural language
